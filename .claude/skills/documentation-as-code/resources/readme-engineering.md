# README Engineering

Guide to creating effective project READMEs.

## Template

```markdown
# Project Name

Brief description (1-2 sentences)

## Features
- Feature 1
- Feature 2

## Quick Start
```bash
npm install
npm run dev
```

## Installation
Detailed steps

## Usage
Examples

## Configuration
Environment variables

## API Reference
Link to full docs

## Contributing
Guidelines

## License
MIT
```

## Badges

```markdown
![Build Status](https://github.com/user/repo/workflows/CI/badge.svg)
![Coverage](https://codecov.io/gh/user/repo/branch/main/graph/badge.svg)
![npm](https://img.shields.io/npm/v/package)
```

## Best Practices

✅ Clear quick start (<5 minutes)
✅ Working code examples
✅ Link to full documentation
✅ Badges for build status
✅ Screenshots for UI projects
